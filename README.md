## IslamicHub

**IslamicHub** is a comprehensive, open-source Islamic education and utility platform designed to help users learn and
live the **Five Pillars of Islam**. It combines in-depth scholarly guides grounded in the Qur'an and Hadith with
high-utility modern tools like a Zakat calculator, prayer time engine, and inheritance (Fara'id) calculator.

---

## 🌟 Features

### 📖 The Five Pillars

Dedicated educational modules for each pillar, featuring authentic citations and scholarly conditions:

* **Faith (Shahada):** Deep dive into the declaration of faith.
* **Prayer (Salah):** Practical guides and spiritual insights.
* **Zakat:** Detailed breakdown of obligatory almsgiving.
* **Fasting (Sawm):** Wisdom and rules for Ramadan.
* **Hajj:** A step-by-step guide to the pilgrimage rites.

### 🛠️ Interactive Tools

* **Zakat Calculator:** Supports multi-asset calculation (Cash, Gold, Silver, Investments) with **live market prices**
  and dual Nisab methods.
* **Prayer Times:** Astronomical calculator with five major calculation methods, browser geolocation, and real-time
  countdowns.
* **Inheritance (Fara'id) Calculator:** A complex engine covering 13 heir types, automatic blocking (hijab) rules, and
  fractional shares based on Qur'an 4:11–12.
* **Qibla Finder:** Great-circle bearing to Makkah with an animated compass interface.
* **Fasting Tracker:** Real-time progress bars for Suhoor and Iftar, plus a Ramadan countdown.

---

## 🎨 Design & Aesthetics

The platform uses a "Modern Islamic" aesthetic, blending traditional geometry with clean UI:

* **Typography:** Uses **Amiri** for elegant Arabic Naskh-style rendering and **Cormorant Garamond** for Latin serif
  headings.
* **Ornamentation:** Features custom Kufic dividers, 8-pointed geometric star backgrounds, and calligraphy frames.
* **Experience:** Smooth SPA transitions, scroll-triggered reveal animations, and a custom "Bismillah" preloader.

---

## 💻 Tech Stack

* **Frontend:** Vanilla JavaScript (ES6+), HTML5, CSS3.
* **Routing:** Custom hash-based Single Page App (SPA) router.
* **External APIs:** * `metals.live` for live gold/silver prices.
    * `frankfurter.app` for currency exchange rates.
    * `openstreetmap.org` for reverse geocoding.
* **Deployment:** Optimized for Netlify with custom security headers and SPA redirects.

## 🚀 Getting Started

### Prerequisites

A modern web browser and a local web server (like VS Code Live Server or Python `http.server`).

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/islamichub.git
   ```
2. Navigate to the directory:
   ```bash
   cd islamichub
   ```
3. Open `index.html` via your local server.
